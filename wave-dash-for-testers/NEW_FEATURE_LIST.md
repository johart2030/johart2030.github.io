# Wave Dash Feature Roadmap

This list keeps the original ideas, but organizes them by current status so it is easier to track what is already in the game and what still needs work.

## Done

1. Easy / Medium / Hard mode
2. Playable on phone / mobile
3. Show total online players live
4. Hovering over the online count shows who is online
5. Custom maps do not give points
6. Garage / shop preview so you can see what you are buying or equipping
7. Full-screen mode
8. Issues page link
9. Random music
10. Public leaderboard visible to all players, including signed-out players
11. Basic multiplayer player rendering:
Other players show their sprite, trail, and color, and they are drawn slightly transparent.
12. Dev-only sprite, trail, and color support
13. Account button polish:
Signed-in users show a profile image or initial.
Signed-out users see a normal Login button.
14. Keyboard support for `ArrowUp` and `W` in addition to `Space`
15. Click counter and FPS counter

## In Progress

1. Fix multiplayer
When the owner starts the game, everyone should start cleanly together.
Other player visibility and tracking still need polishing.
Seed handling should be double-checked.
Obstacle reload behavior in public and private rooms should feel as reliable as singleplayer.

2. Public room player tracking
Public multiplayer should show other players by their real x and y progress, not a mismatched x-axis approximation.

3. Level editor upgrades
Add more editor controls like speed-related objects.
Make the editor easier to use and closer to a real level editor flow.

## Planned

1. Ads
Still being considered.

2. Player levels / XP

3. Add support for more input devices:
Finish the rest of keyboard support and add Sayo device input.

4. Achievements

5. Cache the game for offline play

6. Backgrounds to buy

## Notes

1. Multiplayer is the biggest remaining system and should stay the main focus until it feels fully stable.
2. UI polish and progression expansion should come after multiplayer reliability.
